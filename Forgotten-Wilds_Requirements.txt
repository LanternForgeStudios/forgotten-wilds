You are Claude Code. Build a playable browser-based JRPG MVP called **Forgotten Wilds**.

The game should be a single-player JRPG with live multiplayer town/lobby foundations. It will be hosted on **GitHub Pages** with a **Firebase backend**.

## Core Vision

Create a browser-based online JRPG inspired by classic Final Fantasy-style adventures, set on an alternate North American continent called **Mytherra**. Players are members of the **Lantern Keepers**, an order of explorers, historians, mediators, and protectors who restore harmony between humanity and ancient spirits after an event known as **The Great Silence**.

The MVP should feel like a polished vertical slice: one town, one overworld region, one dungeon, one boss, NPCs, quests, inventory, equipment, turn-based combat, saving, and Firebase persistence.

## Tech Requirements

Use a modern browser-friendly stack:

* Frontend: React + TypeScript + Vite
* Styling: CSS modules or plain CSS
* State management: lightweight React state/context or Zustand
* Backend: Firebase

  * Firebase Auth with anonymous sign-in
  * Firestore for player saves, town presence, player profiles, inventory, and progression
  * Firebase Hosting optional, but project should also be deployable to GitHub Pages
* No paid services required
* Code should be clean, modular, and documented

Create the full project structure, all source files, configuration files, and clear setup instructions.
