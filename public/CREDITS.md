# Asset Credits

All current game assets are MVP **placeholders** (see `status` in `src/assets/registry.ts`) and are expected to
be replaced with final painterly-pixel-art in a later pass. This file tracks attribution for third-party
placeholder assets currently in use.

## Kenney.nl (CC0 1.0 Universal)

The following packs are CC0 — no attribution is legally required, but Kenney's work is excellent and worth
crediting and supporting (kenney.nl):

- **Tiny Dungeon** — `public/assets/tilesets/tiny-dungeon.png` — https://kenney.nl/assets/tiny-dungeon
- **Fantasy UI Borders** — `public/assets/ui/panel-border-000.png`, `panel-border-004.png` —
  https://kenney.nl/assets/fantasy-ui-borders

## Generated placeholders

All portraits, character/enemy sprites, item/equipment/currency icons, and battle backgrounds not listed above
are simple procedurally-generated SVG placeholders (colored panel + label), created for this project. They
carry no external license and are freely replaceable — see the `notes` field on each entry in
`src/assets/registry.ts` for details on what final art should replace them with.
